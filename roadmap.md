# フロントエンド学習ロードマップ

## 目的
- TypeScript + React のスキルをしっかり身につける
- コード読解力・リファクタリング力・設計理解力を向上させる
- 中規模プロジェクトの開発に自信を持つ

## ステージ1：基礎
- [ ] TypeScript の基本と応用型
- [ ] React の基本（コンポーネント、props、state、hooks）
- [ ] JSX / TSX の構造理解
- [ ] プロジェクトの基本設定（Vite、npm）

## ステージ2：応用練習
- [ ] 小～中規模の OSS React プロジェクトをクローンして読む
- [ ] コードの可読性・保守性を意識したリファクタリング
- [ ] Jest を使った単体テストの作成
- [ ] Context Provider やカスタムフックの理解

## ステージ3：中規模プロジェクト
- [ ] 用意されたバックエンド(API) を使ったフロントエンド作成
- [ ] 外部 API の連携（Spotify 風など）
- [ ] ルーティング・状態管理（必要に応じて Redux）
- [ ] テスト・ドキュメントの追加

## ステージ4：ポートフォリオ用プロジェクト
- [ ] GitHub 用に完成度の高いプロジェクトを作成
- [ ] リファクタリング前後の比較を含める
- [ ] 学習で得た知見や設計判断をまとめる
